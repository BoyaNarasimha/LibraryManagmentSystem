# 📚 Library Management System

A web-based **Library Management System** built using **Java, JSP, Servlets, JDBC, and Oracle Database**.
This project helps manage books efficiently with features like authentication, CRUD operations, and search functionality.

---

## 🚀 Features

* 🔐 User Login & Registration
* 📊 Dashboard with operations
* ➕ Add new books
* ✏️ Update book details
* 🗑️ Delete books
* 📖 View all books
* 🔍 Search books by title or author
* 🎨 Responsive UI using Bootstrap

---

## 🛠️ Technologies Used

* **Frontend:** HTML, CSS, Bootstrap, JSP
* **Backend:** Java Servlets
* **Database:** Oracle (JDBC)
* **Server:** Apache Tomcat

## ⚙️ Setup & Run

1. Clone the repository

   ```
   git clone https://github.com/BoyaNarasimha/LibraryManagmentSystem.git
   ```

2. Import project into Eclipse

3. Configure Apache Tomcat Server

4. Update database credentials in:

   ```
   com.util.DBConnection.java
   ```

5. Run project on server

6. Open in browser:

   ```
   http://localhost:8080/Library_Managment_System/
   ```
